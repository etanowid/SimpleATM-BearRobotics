Simple ATM program for Bear Robotics (python)

Clone this project using git clone https://github.com/etanowid/SimpleATM-BearRobotics.git
or 
Download the zip file and unzip it

Run the program:
cd into the directory where you place the repository
> python3 main.py


The example user is Betty Beep Boop with pin number = 1234
New user can be added in main.py
